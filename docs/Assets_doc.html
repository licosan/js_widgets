Summary

Assets aims to be a minimal, lightweight, assets manager. Assets can be Images, CSS files or HTML fragments.
It just makes it easier to use or reuse any assets, in a centralized & consistent way.
It also stores everything loaded in a central storage to avoid loading an asset twice from the network when reusing.

Basic usage

HTML
<div class="rouge">This is red if the asset style is loaded.</div> <div id="logo"></div> <div id="fillin" style="border:1px solid black;width:50%;">Box empty so far...</div>
Javascript
var rn; document.addEventListener('DOMContentLoaded', function () { // Load and apply the stylesheet called "test.css" (in assets/css) window.Widgets.Assets.LoadCss('test'); // Load the image called "Nicsys.jpg" (in assets/images), and add it to the #logo div window.Widgets.Assets.appendImage('Nicsys.jpg', '#logo'); // Load the html fragment called "loreipsum.html" (in assets/html) anduse it as content for the #fillin div window.Widgets.Assets.replaceHtml('loremipsum','#fillin'); }, false);

Assets API:

Assets.LoadCss('name')

Loads a stylesheet, and apply it as soon as it is available.
Parameters:
  • Stylesheet name (with or without the ".css")

Assets.LoadHtml(name, callback)

Loads an html fragment, and calls the callback with the html as parameter it as soon as it is available.
Parameters:
  • HTML fragment name (with or without the ".html")
  • callback (with the html string as only parameter)

Assets.replaceHtml(name, selector)

Loads an html fragment, and replaces the content of the target element with it.
Parameters:
  • HTML fragment name (with or without the ".html")
  • target element(s) selector

Assets.appendHtml(name, selector)

Loads an html fragment, and appends the content at the end of the target element with it.
Parameters:
  • HTML fragment name (with or without the ".html")
  • target element(s) selector

Assets.prependHtml(name, selector)

Loads an html fragment, and appends the beginning at the end of the target element with it.
Parameters:
  • HTML fragment name (with or without the ".html")
  • target element(s) selector

Assets.GetImage(name)

Loads an image, and returns an image node.
If you call it again on the same image, it will clone the original node and return the clone (without reloading the image), so you could use the image a two separate plaaces..
Parameters:
  • Image name

Assets.appendImage(name, selector)

Loads an image, and appends it at the end of the target element.
Parameters:
  • Image name
  • target element(s) selector

Assets.prependImage(name, selector)

Loads an image, and appends it at the beginning of the target element.
Parameters:
  • Image name
  • target element(s) selector
