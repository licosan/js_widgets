Summary

Quick and dirty <ul> reactive list.
Mainly to be used as as real widget example in other tests.

Basic usage

HTML
<div id="listbox" style="border: 1px solid blue;width: 30%;">Here comes the list...</div>
Javascript
var Users = [ { "id": "32164", "name": "Toto" }, { "id": "78645", "name": "Tintin" }, { "id": "49642", "name": "Milou" }, { "id": "75244", "name": "Sheldon" }, { "id": "45668", "name": "Leonard" }, ]; document.addEventListener('DOMContentLoaded', function () { Widgets.loadWidget('DynaList', function() { usr_lst = new Widget( 'DynaList', { 'selector': '#listbox', 'source': Users, 'listid':'usrlst', 'entry_id': 'id', 'entry_label': 'name', }); }); });

DynaList API:

Instanciation :

Parameters:
  • container: list parent (takes precedence over selector if both present)
  • selector: list parent selector
  • source: the array containing the list elements
  • listid: Both the DOM id of the <ul> and the suffix of the variable used in Picoreac
  • entry_id: Each <li> will have an "item_id" attribute, with the value of the entry_id named property for this array entry
  • entry_label: The text of each <li> will picked from the value of the entry_label named property for this array entry
